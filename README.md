# Task Manager

A simple task management app with add, delete, and mark complete features.

## Setup Instructions

### Frontend

1. Navigate to the `frontend` directory:
   ```bash
   cd frontend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```

### Backend

1. Navigate to the `backend` directory:
   ```bash
   cd backend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the server:
   ```bash
   npm start
   ```

### Environment Variables

Create a `.env` file in the `backend` directory based on the `.env.example` file.

## Features

- Add new tasks
- Delete tasks
- Mark tasks as complete
- View all tasks

## Technologies Used

- Frontend: React, Tailwind CSS
- Backend: Node.js, Express.js
- Database: SQLite
